# workflow

A fullstack Project Management Software made using React JS (Typescript), Node JS + Express (Typescript), and Mongo DB.

<img src="https://user-images.githubusercontent.com/63698375/161473461-14582240-ad87-4c2d-8395-3d44c3879739.png" alt="boardDetailPage" style="width:100%;"/>

## Technologies Used:

 ### Frontend

  - React JS + Typescript
  - Redux Toolkit
  - React Query
  - Tailwind CSS
  - Formik + Yup
  - React Beautiful DND
  - React Toastify
  - React Select
  
### Backend

 - Node JS + Express + Typescript
 - Mongo DB (with Mongoose)
 - JWT tokens for authentication
 - Google Auth
 - Node Mailer
 
 ## Features
 
 - Signup and Login
 - HomeFeed (dedicated for both Loggedin users and anonymous users)
 - Pre defined Communities
 - Create, Read, and Delete Posts
 - Create, Read, and Delete Comments
 - Upvote and Downvote Posts
 - Search Posts
 - Sorting
 - Filter Posts based on Community/User.
 - JWT based auth (both accessToken and refreshToken)
